# UX Copywriting: Glossary & Dictionary

## Core Terminology (RU/EN)

| English Term | Russian Term | Context |
| :--- | :--- | :--- |
| **Workspace / Tenant** | Организация / Пространство | The top-level container for all data. |
| **Team Member** | Участник команды | A person who has access to the workspace. |
| **Material** | Материал | A resource used in construction/works. |
| **Work** | Работа | A specific task or assembly in an estimate. |
| **Estimate** | Смета | The document calculating the cost of works and materials. |
| **Phase** | Этап / Фаза | A logical group of works in an estimate. |
| **Sharing** | Поделиться / Доступ | Granting access to an estimate via a link. |
| **Project** | Проект | The high-level container for estimates related to a specific client. |
| **Impersonation** | Имперсонация | Admin acting on behalf of a user for support. |

## Tone & Voice Guidelines
- **Be Concise:** Use as few words as possible without losing meaning.
- **Be Helpful:** Errors should explain *why* something happened and *how* to fix it.
- **Direct Address:** Use "Вы" (Polite/Formal) or "Ваш" in Russian.
- **No Jargon:** Avoid "Database Error", "404", "Exception".

## Message Templates

### Error Messages
- **Access Denied:** "У вас нет прав для выполнения этого действия. Обратитесь к администратору организации."
- **Generic Error:** "Произошла непредвиденная ошибка. Мы уже работаем над её исправлением."
- **Not Found:** "Запрашиваемый ресурс не найден. Возможно, он был удален или перемещен."

### Notifications
- **Invitation:** "[Имя] пригласил вас в команду [Организация]."
- **Estimate Shared:** "Смета '[Название]' доступна для просмотра по прямой ссылке."

## Admin Specifics
- **Action Buttons:** Use verbs (Удалить, Сохранить, Применить).
- **Confirmation Dialogs:** Always ask before destructive actions (e.g., "Вы уверены, что хотите удалить этот материал?").
